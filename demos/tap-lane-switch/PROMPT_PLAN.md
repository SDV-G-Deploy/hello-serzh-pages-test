# PROMPT PLAN — Tap Lane Switch (mobile-first MVP)

## Goal
Сделать лёгкое 45-секундное демо для одной руки: игрок циклически переключает 3 дорожки, избегает препятствий и собирает монеты.

## Core loop
- Игрок стартует забег.
- По трем lane сверху падают объекты двух типов: obstacle и coin.
- Тап по кнопке «Сменить lane» циклично переключает позицию игрока.
- За уклонение и сбор монет начисляются очки.
- Сессия 45 секунд, затем перезапуск.

## Scoring
- Coin: +2.
- Успешно пропущенное препятствие: +1.
- Столкновение с препятствием: -1 (не ниже 0), combo reset.
- Каждые 7 combo: +1 бонус.

## Mobile-first constraints
- Экран 360–430px приоритетно.
- Только один основной action button (one-thumb input).
- Time-to-play < 10 секунд.
- Без библиотек, чистый HTML/CSS/JS.

## Done criteria
- Playable loop 45s + restart.
- UI: timer, score, combo, best.
- Ясный фидбек по событиям (coin/hit).
- Стабильная работа на мобильном без лагов.
