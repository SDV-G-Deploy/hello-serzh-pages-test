# Notes update template

`notes.html` рендерит контент из `notes-data.js`.

## Как добавить новую запись дня

1. Открой `notes-data.js`.
2. В массив `entries` добавь объект **в начало** (новые сверху):

```js
{
  date: '2026-03-14',
  title: 'Короткий заголовок дня',
  context: 'С чего начинали и почему это было важно.',
  actions: [
    'Что конкретно сделали (1)',
    'Что конкретно сделали (2)'
  ],
  result: 'Какой итог получили к концу дня.'
}
```

## Как обновлять insights

В `notes-data.js` есть 3 блока:
- `insights.worked` — что сработало
- `insights.failed` — что не сработало
- `insights.next` — что пробуем дальше

Добавляй короткие, конкретные пункты (1 мысль = 1 строка).
